Event Management System

This project is a desktop application developed in Java using Swing for the GUI. 
It is designed to manage events, users, registrations, and tickets in an organized way.

Features:
- User Registration and Login with validation.
- Event management: view available events, register for events.
- Ticket generation with unique ticket numbers.
- Admin functionalities: manage users, generate reports (events summary, user statistics, attendance, revenue).
- Send ticket via email (simulation in UI).

Technologies & Tools Used:
- Java SE
- Swing (GUI)
- JDBC (Database Connectivity)
- SQLite / MySQL (Database)
- SQL queries for CRUD operations
- Validation utils for email and password formats
- PreparedStatements to prevent SQL injection
